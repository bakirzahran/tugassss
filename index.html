Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

1. CSS CSS adalah seperangkat alat untuk membuat dan memodifikasi jenis huruf, peringatan, warna, bentuk, dan elemen terkait gaya lainnya yang tidak dapat dilakukan menggunakan elemen HTML. Bisa dikatakan bahwa CSS adalah kepanjangan dari HTML untuk pemformatan dokumen web. CSS apa pun memudahkan Anda membuat dan memelihara situs web karena CSS mengintegrasikan presentasi dan kode web yang baru dibuat. 2. JavaScript JavaScript adalah bahasa pemrograman yang paling umum digunakan untuk membuat aplikasi berbasis web. Bahasa ini dapat membuat website lebih logis -- membuatnya lebih interaktif. JavaScript memberi developer kemampuan untuk menerapkan fitur kompleks pada halaman web. Setiap kali halaman web menampilkan informasi statis untuk pengguna, seperti pembaruan konten secara live, peta interaktif, animasi 2D/3D, slider video/musik, dan lain-lain, dapat dipastikan bahwa web tersebut menggunakan JavaScript.

Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

1. CSS Menggunakan CSS (Cascading Style Sheets), kita dapat menerapkan gaya ke halaman web kita agar terlihat persis seperti yang kita inginkan. Ini berfungsi karena CSS terhubung ke DOM (Document Object Model). Dengan CSS dan integrasinya dengan DOM, kita dapat dengan cepat dan mudah menata ulang elemen apa pun. Cara untuk mengintegrasikan CSS ke halam web ada 2 cara yaitu eksternal dan internal.Eksternal dengan membuat sebuah file CSS kemudian membuat link dalam file html ( link) kemudian Di dalam text editor, simpan file menggunakan nama css.html, pastikan tidak ada spasi atau karakter lain dalam nama file. File harus disimpan di dalam root dokumen Anda. sedangkan secara internal dengan menggunakan tag style dalam halaman HTML dan memanggil satu persatu tag yang ingin di desain. 2. Javascript memasukkan JavaScript ke dalam halaman. kita dapat menambahkan JavaScript langsung ke file HTML, mereferensikan file JavaScript terpisah, atau melakukan keduanya — dan kami membantu kita memahami kapan setiap opsi sesuai. Terdapat 2 cara yaitu eksternal dan internal. Cara eksternal enambahkan JavaScript adalah dengan menggunakan file JavaScript eksternal kemudian menambahkan JavaScript eksternal dengan menggunakan atribut src <=script type=”text/javascript” src=”externalfile.js”script sedangkan cara internal yaitu Menambahkan tag JavaScript, menambahkan JavaScript ke halaman dengan tag <=script>
